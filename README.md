Portfolio personal desarrollado con React que muestra mi trayectoria como Técnico Audiovisual y desarrollador asistido por IA. Diseño moderno, oscuro y con animaciones sutiles.

🚀 Demo en vivo
🔗 Ver portfolio online (https://incongo.github.io/mi-portfolio/)

✨ Características principales

Sección Descripción
Header Navegación sticky con logo animado (corchetes que se abren) y enlaces sociales
Hero Presentación personal con diseño limpio y llamado a la acción
Trayectoria Timeline interactiva con efecto reveal al hacer scroll
Proyectos Grid de proyectos con imágenes, miniaturas y modal para vista ampliada
Contacto Enlaces directos a email, GitHub, LinkedIn e Instagram
Fondo dinámico Logos de tecnologías flotando con movimiento suave

Frontend

⚙️ Instalación y uso local

# Clonar el repositorio

git clone https://github.com/Incongo/mi-portfolio.git
cd mi-portfolio

# Instalar dependencias

npm install

# Iniciar servidor de desarrollo

npm run dev

# Construir para producción

npm run build

# Previsualizar la build

npm run preview

📝 Personalización
Para personalizar este portfolio:

Datos personales: Edita Hero.jsx y Contacto.jsx

Trayectoria: Modifica el array experiencia en Experiencia.jsx

Proyectos: Actualiza el array proyectos en ProyectosDestacados.jsx

Logos: Añade/elimina logos en FondoLogos.jsx

Colores: Cambia #4a9eff por tu color en los archivos CSS
